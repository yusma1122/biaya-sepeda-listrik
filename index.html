üìã Spesifikasi E-bike

üí° Apa itu Voltase?

Voltase adalah "kekuatan listrik" baterai e-bike Anda. Seperti tekanan air dalam selang - semakin tinggi voltase, semakin kuat tenaga motor. Biasanya: 36V (standar), 48V (sedang), 60V+ (kencang). Cek stiker di baterai atau buku manual.

üí° Apa itu Kapasitas (Ah)?

Ah = Ampere Hour, artinya "berapa lama baterai bisa menyuplai listrik". Ibarat tangki bensin - semakin besar Ah, semakin jauh bisa jalan. Contoh: 10Ah (jarak pendek), 20Ah (jarak sedang), 30Ah+ (jarak jauh). Tertulis di baterai atau spesifikasi.

üí° Apa itu Range?

Range = jarak tempuh sekali charge penuh. Seperti motor bensin dengan tangki penuh bisa berapa km. Biasanya tertulis di brosur (contoh: "60km sekali charge"). PENTING: ini jarak ideal - di praktik bisa lebih pendek tergantung jalanan dan cara berkendara.

üí° Apa itu Tarif Listrik?

Ini harga listrik PLN per kWh (kilo Watt hour). Seperti harga bensin per liter, tapi untuk listrik. Cek tagihan listrik rumah Anda, biasanya tertulis "Rp/kWh". Standar PLN: Rp 1.467/kWh (2025). Kalau tidak tahu, pakai angka default ini.

üöó Pola Penggunaan

üí° Jarak yang Anda tempuh setiap hari

Contoh: rumah ke kantor PP = 15km, berarti total harian 30km. Atau rumah-pasar-sekolah-rumah = 25km. Hitung semua perjalanan dalam 1 hari, bukan sekali jalan saja.

üí° Berapa hari Anda pakai e-bike dalam seminggu?

Contoh: kerja Senin-Sabtu = 6 hari. Atau cuma weekdays = 5 hari. Kalau kadang libur atau WFH, kurangi jadi 4-5 hari. Ini untuk hitung biaya yang realistis.

üí° Cara Anda berkendara sehari-hari

Eco: Pelan-pelan, hemat baterai, cocok jarak jauh
Normal: Standar, seimbang kecepatan & hemat
Sport: Ngebut, boros baterai tapi cepat sampai

üí° Seperti apa jalanan yang biasa Anda lewati?

Datar: Jalan lurus, tidak ada tanjakan (hemat baterai)
Campuran: Ada naik-turun, tapi tidak terlalu curam
Berbukit: Banyak tanjakan curam (boros baterai)

üí∞ Analisis ROI & Balik Modal

üí° Total investasi awal untuk beli e-bike

Masukkan harga beli e-bike lengkap dengan aksesoris. Contoh: Rp 15.000.000 untuk e-bike kelas menengah, atau Rp 8.500.000 untuk entry level.

üí° Berapa Anda habiskan bensin motor sekarang?

Hitung total biaya bensin motor per minggu. Contoh: isi 3x @Rp 50.000 = Rp 150.000/minggu. Atau kalau isi harian Rp 20.000 x 7 = Rp 140.000/minggu.

Penjelasan detail setiap langkah perhitungan dengan contoh menggunakan data default:

1Ô∏è‚É£ Menghitung Energi Baterai (kWh)

Rumus: Energi (kWh) = Voltase √ó Kapasitas √∑ 1000

üìç Dari mana angkanya:

‚Ä¢ Voltase (48V) = Input Anda di "Voltase Baterai"

‚Ä¢ Kapasitas (10Ah) = Input Anda di "Kapasitas Baterai"

‚Ä¢ √∑ 1000 = Konversi dari Watt hour (Wh) ke kiloWatt hour (kWh)

Contoh Perhitungan:

48V √ó 10Ah √∑ 1000 = 0.48 kWh

üí° Artinya: Baterai Anda bisa menyimpan 0.48 kWh energi listrik (seperti tangki bensin 0.48 liter)

2Ô∏è‚É£ Menghitung Biaya Charging

Rumus: Biaya = (Energi √∑ Efisiensi Charging) √ó Tarif PLN

üìç Dari mana angkanya:

‚Ä¢ Energi (0.48 kWh) = Hasil dari langkah 1

‚Ä¢ Efisiensi Charging (0.92 = 92%) = Standar industri (charger tidak 100% efisien)

‚Ä¢ Tarif PLN (Rp 1.467) = Input Anda di "Tarif Listrik"

Mengapa ada Efisiensi 92%?

Saat ngecas, tidak semua listrik dari PLN masuk ke baterai. Ada yang hilang jadi panas di charger (sekitar 8%). Jadi kalau baterai butuh 0.48 kWh, PLN harus supply 0.52 kWh.

Contoh Perhitungan:

(0.48 √∑ 0.92) √ó Rp 1.467 = 0.52 kWh √ó Rp 1.467 = Rp 765

üí° Artinya: Sekali ngecas full habis Rp 765 listrik

3Ô∏è‚É£ Menghitung Jarak Real (Praktis)

Rumus: Jarak Real = Range √ó Efisiensi Mode √ó (1 √∑ Faktor Medan)

üìç Dari mana angkanya:

‚Ä¢ Range (40 km) = Input Anda di "Range per Charge" (jarak ideal pabrik)

‚Ä¢ Efisiensi Mode (0.9 = 90%) = Pilihan Anda di "Mode Berkendara"

‚Ä¢ Faktor Medan (1.0) = Pilihan Anda di "Kondisi Medan"

Mengapa jarak berkurang?

‚Ä¢ Mode Eco (90%) = Hemat, tapi tetap ada loss 10%

‚Ä¢ Mode Normal (80%) = Standar, loss 20%

‚Ä¢ Mode Sport (70%) = Ngebut, boros 30%

‚Ä¢ Medan Datar (1.0x) = Normal

‚Ä¢ Medan Campuran (1.2x) = Baterai 20% lebih boros

‚Ä¢ Medan Berbukit (1.5x) = Baterai 50% lebih boros

Contoh Perhitungan:

40 km √ó 0.9 √ó (1 √∑ 1.0) = 36 km real

üí° Artinya: Yang tertulis 40km di brosur, praktiknya cuma 36km

4Ô∏è‚É£ Menghitung Biaya per km

Rumus: Biaya per km = Biaya Charge √∑ Jarak Real

üìç Dari mana angkanya:

‚Ä¢ Biaya Charge (Rp 765) = Hasil dari langkah 2

‚Ä¢ Jarak Real (36 km) = Hasil dari langkah 3

Contoh Perhitungan:

Rp 765 √∑ 36 km = Rp 21,26 per km

üí° Artinya: Setiap 1 km jalan, biaya listrik Rp 21 (vs bensin motor ~Rp 500-800/km)

5Ô∏è‚É£ Menghitung Biaya Bulanan

Rumus: Biaya Bulanan = Biaya per km √ó Jarak Harian √ó Hari Aktif √ó (52√∑12)

üìç Dari mana angkanya:

‚Ä¢ Biaya per km (Rp 21,26) = Hasil dari langkah 4

‚Ä¢ Jarak Harian (20 km) = Input Anda di "Jarak Harian"

‚Ä¢ Hari Aktif (5 hari) = Input Anda di "Hari Aktif per Minggu"

‚Ä¢ (52√∑12 = 4.33) = Konversi minggu ke bulan (52 minggu/tahun √∑ 12 bulan)

Contoh Perhitungan:

Rp 21,26 √ó 20 km √ó 5 hari √ó 4.33 = Rp 9.213 per bulan

6Ô∏è‚É£ Menghitung ROI Balik Modal

Rumus: Waktu Balik Modal = Harga E-bike √∑ Penghematan per Bulan

üìç Dari mana angkanya:

‚Ä¢ Harga E-bike (Rp 15.000.000) = Input Anda di "Harga Beli E-bike"

‚Ä¢ Biaya Motor per Bulan = Bensin per Minggu √ó (52√∑12) = Rp 150.000 √ó 4.33 = Rp 650.000

‚Ä¢ Biaya E-bike per Bulan (Rp 9.213) = Hasil dari langkah 5

‚Ä¢ Penghematan = Rp 650.000 - Rp 9.213 = Rp 640.787 per bulan

Contoh Perhitungan:

Rp 15.000.000 √∑ Rp 640.787 = 23.4 bulan

üí° Artinya: Investasi e-bike balik modal dalam 23.4 bulan (hampir 2 tahun)
